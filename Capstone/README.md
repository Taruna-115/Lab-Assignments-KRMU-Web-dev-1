**PayFlex Landing Page**
A clean and responsive Payflex landing page built using HTML5 and CSS3.
This project showcases modern UI design, semantic structure, and responsive layouts without using JavaScript.

**Overview**
PayFlex is a conceptual digital payment platform.
This landing page highlights the product’s features, pricing, testimonials, and overall value proposition in a simple, visually appealing layout.
All styling and layout behavior are implemented using pure CSS (Flexbox + Grid).

**Objectives**
This project was created to:
1. Demonstrate a multi-section website layout using only HTML & CSS
2. Apply semantic HTML5 for accessibility and structure
3. Build a fully responsive design suitable for mobile, tablet, and desktop
4. Present information clearly using modern UI sections like hero banners, feature cards, pricing tables, and testimonials
5. Fulfill all instructions and required sections from the assignment document (DOCX)

**Features**
1. Responsive hero section with headline, description, and call-to-action buttons
2. Sticky header navigation
3. Feature highlights presented using a grid system
4. “How it Works” step-by-step section
5. Pricing plans with clean card layout
6. Testimonials using blockquote styling
7. Structured footer with navigation and company details
8. Fully responsive layout using media queries
9. Minimal, clean, and scalable CSS
