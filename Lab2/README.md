**Lab2- EXTERNAL CSS STYLING** 
  
**OVERVIEW**

This lab focuses on applying external CSS to enhance the visual presentation and structure of the html file created in Lab 1.
The goal is to focus on using different CSS selectors,the box-model,typography, and positioning techniques.

**OBJECTIVE**
1. Link and apply an external CSS stylesheet to existing Lab 1 HTML file.
2. Develop a consistent color theme using HEX or RGB color values.
3. Customize typography with Google Fonts.
4. Style elements using tag type, class, and ID selectors.
5. Customize buttons using border-radius,background color,and padding.
6. Style tables to display technical skills or tools.
7. Implementing various CSS positioning techniques- Fixed,Static.
8. Apply the CSS box model to control spacing and layout.

**TECHNOLOGIES USED**
1. HTML5
2. CSS
3. Google Fonts
